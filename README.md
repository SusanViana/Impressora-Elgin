# Impressora-Elgin
<h1>Sistema de Impressão com DLL Elgin<h1>

Um sistema em Java que utiliza a DLL da Elgin para abrir conexão com a impressora, realizar recuperação (texto, QR Code, código de barras, XML de SAT), emitir sinal sonoro e abrir gavetas, integrado a um menu via switch case.

<h1>Objetivo :<h1>
Abrir e fechar conexão com impressoras Elgin.
Enviar comandos de impressão (texto, QR Code, código de barras).
Imprimir XML de SAT e XML de cancelamento SAT.
Emitir sinal sonoro.
Abrir gaveta da impressora.
Controlar o fluxo via switch case, simulando um mini PDV.

<h1>Como executar:<h1>
Instale o Java (versão 17 ou superior).
Baixe ou coloque uma DLL da Elgin na pasta do projeto.
Abra o projeto no IntelliJ IDEA .
Ajuste os caminhos dos arquivos XML, se necessário.
Execute o arquivo Main.java .
Utilize o menu para escolher a função (abrir conexão, imprimir, etc.).

<h1>Tecnologias usadas:<h1>
Java
JNA 
DLL da Elgin
IntelliJ IDEA
