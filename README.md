# Impressora-Elgin
<h1>Sistema de Impressão com DLL Elgin<h1>

<p>Um sistema em Java que utiliza a DLL da Elgin para abrir conexão com a impressora, realizar recuperação (texto, QR Code, código de barras, XML de SAT), emitir sinal sonoro e abrir gavetas, integrado a um menu via switch case.</p>

<h1>Objetivo :<h1>
<p>Abrir e fechar conexão com impressoras Elgin.</p>
<p>Enviar comandos de impressão (texto, QR Code, código de barras).</p>
<p>Imprimir XML de SAT e XML de cancelamento SAT.</p>
<p>Emitir sinal sonoro.</p>
<p>Abrir gaveta da impressora.</p>
<p>Controlar o fluxo via switch case, simulando um mini PDV.</p>

<h1>Como executar:<h1>
<p>Instale o Java (versão 17 ou superior).</p>
<p>Baixe ou coloque uma DLL da Elgin na pasta do projeto.</p>
<p>Abra o projeto no IntelliJ IDEA.</p>
<p>Ajuste os caminhos dos arquivos XML, se necessário.</p>
<p>Execute o arquivo Main.java.</p>
<p>Utilize o menu para escolher a função (abrir conexão, imprimir, etc.).</p>

<h1>Tecnologias usadas:<h1>
<p>Java</p>
<p>JNA </p>
<p>DLL da Elgin</p>
<p>IntelliJ IDEA</p>
